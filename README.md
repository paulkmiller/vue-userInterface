# Vue User Interface

* [Vue Docs](https://vuejs.org/v2/guide/)
* [Vuetify Docs](https://vuetifyjs.com/en/components/calendars/)

## Setup
```shell
git clone https://github.com/paulkmiller/vue-userInterface.git
```

### Installation

```shell
cd vue-userInterface

npm install

cd ../api

npm install
```

### Project Startup
```shell
cd vue-userInterface

cd api

npm run serve 
```
